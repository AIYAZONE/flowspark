# 设计资产索引

## 组件
- 仪表盘：核心行动卡、进度条、评分组件、趋势图容器
- 目标列表：目标卡片、优先级 Badge、进度条、分页与筛选
- 目标详情：信息分组卡、行动列表、Tab/段落卡
- 今日行动：行动条目（复选框）、完成反馈按钮、空态与骨架
- 通用：导航（侧栏/底部）、表单输入、按钮、Toast、Skeleton

## 图标与插画
- 图标库：Lucide React
- 用例：完成（check）、警告（alert-triangle）、评分（sparkles）、趋势（line-chart）
- 插画：空态与引导页统一风格（极简线稿，低饱和度）

## 配色 Token
- 主色：electric-emerald
- 辅助：holographic-blue、nebula-purple、vitality-orange
- 中性色：ceramic-white、slate-grey、titanium
- 状态：success、warning、error
- 暗/明模式映射：同名 Token 在 dark/light 下取不同色值
