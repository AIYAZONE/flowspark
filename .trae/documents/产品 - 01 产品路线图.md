# 产品迭代路线图：个人目标系统 (Personal Goal System)

## 产品愿景

打造一个结合科学目标管理方法论与 AI 智能辅导的个人成长平台，帮助用户通过“微习惯”建立正向循环，实现自我提升。

## 第一阶段：基础夯实与体验优化 (当前 - 2026 Q1)

**核心关注：稳定性、性能与用户体验打磨**

* **代码质量与测试**：

  * 引入单元测试 (Vitest)：覆盖核心业务逻辑（如评分计算、日期处理）。

  * 添加端到端测试 (Playwright)：确保关键流程（登录 -> 创建目标 -> 打卡）的稳定性。

  * Vercel 部署集成：利用 Vercel 自动部署功能，并配置 Preview Deployments 进行代码预览。

* **用户体验改进**：

  * 增加骨架屏 (Skeleton loaders)：优化数据加载时的视觉体验。

  * 实现“撤销”功能：允许用户撤回误删操作。

  * PWA 配置：添加 `manifest.json` 和 Service Workers，支持应用安装到桌面/手机，提供原生应用般的体验。

* **数据可视化增强**：

  * 升级仪表盘图表：增加周/月维度的趋势分析视图。

## 第二阶段：AI 智能集成 (2026 Q2)

**核心关注：智能辅助与个性化教练**

* **智能目标生成器 (AI Agent)**：

  * *功能*：用户输入模糊的想法（如“我想变健康”），AI 自动拆解为符合 SMART 原则的结构化目标和阶段性里程碑。

  * *技术栈*：OpenAI API / Anthropic Claude, LangChain/Vercel AI SDK。

* **每日行动智能推荐**：

  * *功能*：根据目标优先级、截止日期及用户历史完成率，动态推荐今日最优行动清单。

* **AI 问责教练 (Accountability Coach)**：

  * *功能*：基于对话界面的每日复盘助手。AI 会在每晚询问反思性问题（“今天为什么没完成这个任务？”）并提供鼓励或建议。

  * *技术栈*：RAG (检索增强生成)，基于用户的历史行为数据。

## 第三阶段：社交与游戏化 (2026 Q3)

**核心关注：通过社区力量激发动力**

* **社交功能**：

  * 公开个人主页（可选）。

  * 目标模板分享（社区知识库）。

  * “监督伙伴”系统 (Accountability Buddy)：邀请朋友围观进度，相互监督。

* **游戏化机制**：

  * 成就勋章系统（如“7日连胜”、“目标粉碎者”）。

  * 经验值 (XP) 与等级系统。

  * 排行榜（好友榜/全球榜）。

## 第四阶段：生态扩展 (2026 Q4+)

**核心关注：全场景接入**

* **原生移动应用**：使用 React Native 或 Capacitor 封装，发布 iOS/Android 应用商店版本。

* **第三方集成**：

  * Google Calendar / Outlook 双向同步。

  * Apple Health / Google Fit 数据接入（针对健康类目标自动打卡）。

  * 浏览器插件：支持快速收集灵感和目标。

## 成功度量与风险

- 北极星指标：每日活跃用户（DAU）与“今日行动完成率”
- 关键 KPI：连续打卡7日用户占比、目标达成率、登录成功率
- 风险与依赖：Supabase 服务稳定性、Vercel 预览域名配置、移动端适配成本
- 资源估算：设计/前端/后端投入与周期（按阶段细化）
