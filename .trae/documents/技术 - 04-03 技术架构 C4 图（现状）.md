## 1) C4-Context（系统上下文）

```mermaid
C4Context
title FlowSpark - System Context

Person(user, "用户", "管理目标/行动、打分、使用 AI 辅助")

```

## 2) C4-Container（容器图）

```mermaid
C4Container
title FlowSpark - Container Diagram

Person(user, "用户", "在浏览器中使用 FlowSpark")

Container(web, "FlowSpark Web App", "Next.js 16 / React 19", "App Router + Server Actions + Route Handlers（同一部署单元）")
Container(sw, "PWA Service Worker", "Workbox (next-pwa)", "缓存字体/图片/公开 storage 资源，提升离线与加载体验")

Container_Boundary(supabaseProj, "Supabase Project") {
  Container(auth, "Auth", "Supabase Auth", "注册/登录/会话管理/邮箱验证")
  ContainerDb(db, "Postgres DB", "Supabase Postgres", "user_profiles/goals/actions/daily_scores/xp_logs 等")
  Container(storage, "Storage", "Supabase Storage", "avatars bucket（public read）")
}

Container_Ext(aiApi, "AI Provider API", "OpenAI Compatible", "Chat Completions（DeepSeek/OpenAI）")
Container_Ext(analytics, "Vercel Analytics", "SDK + Ingest", "前端事件采集与统计")

Rel(user, web, "访问与交互", "HTTPS")
Rel(web, sw, "注册与使用缓存", "Service Worker")

Rel(web, auth, "登录/鉴权/交换 session", "supabase-js + @supabase/ssr")
Rel(web, db, "读写业务数据（RLS 约束）", "supabase-js")
Rel(web, storage, "上传/读取头像等对象", "supabase-js")

Rel(web, aiApi, "请求 AI 生成结构化建议", "JSON/HTTPS")
Rel(web, analytics, "上报埋点事件", "SDK")
```

## 3) 对应代码入口
- Web App（页面/布局）：`../../src/app/*`
- Server Actions：`../../src/app/(authenticated)/**/actions.ts`
- API Route Handlers：`../../src/app/api/**/route.ts`
- PWA 缓存策略：`../../next.config.ts` 的 `runtimeCaching`
- Supabase client/middleware：`../../src/lib/supabase/*`
- AI Provider：`../../src/lib/ai/client.ts`
- Analytics：`../../src/lib/analytics.ts`
